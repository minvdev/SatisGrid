# SatisGrid

SatisGrid es una herramienta web para diseñar planos de fábricas en _Satisfactory_ usando una cuadrícula precisa basada en foundations.

El objetivo es sustituir el diseño manual en papel por una interfaz rápida y visual que permita planificar layouts de forma clara antes de construir en el juego.

---

## Características iniciales

- Grid visual donde cada celda representa una foundation
- Colocación de elementos en el plano
- Interfaz simple con panel lateral de componentes
- Edición rápida (añadir / eliminar elementos)

---

## Objetivo del proyecto

Este proyecto está enfocado en:

- Practicar desarrollo frontend con React + TypeScript
- Trabajar con sistemas de coordenadas y estado
- Construir una herramienta real orientada a un caso práctico
